# OnboardingRacialRW
Este repositório documenta a estrutura inicial de onboarding do pilar de equidade racial da Risk Women (RW).  O objetivo do onboarding é mapear, de forma estruturada, quem são as mulheres pretas da comunidade RW.  Mais do que um cadastro, este onboarding funciona como uma base estratégica para direcionar ações futuras do pilar.
